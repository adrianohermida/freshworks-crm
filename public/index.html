Configuração SQL Server Local (Windows 11)

Você importou o repositório, mas ainda não executou o SQL: siga este roteiro e valide no SSMS.

  1. Abra o SQL Server Configuration Manager e habilite TCP/IP.
  2. Reinicie o serviço da instância (SQL Server (SQLEXPRESS ou MSSQLSERVER)).
  3. No SSMS, execute o script de criação abaixo.
  4. Execute a consulta de validação para confirmar se o banco/tabela foram criados corretamente.

Conexão local sugerida

  • Servidor: localhost\\SQLEXPRESS (ou localhost)
  • Porta: 1433 (se TCP/IP habilitado)
  • Banco: AdvocaciaOperacional
  • Autenticação: Windows ou SQL Login

String de conexão (exemplo Node)

DB_CLIENT=sqlserver
DB_HOST=localhost
DB_PORT=1433
DB_NAME=AdvocaciaOperacional
DB_USER=sa
DB_PASSWORD=SuaSenhaForte
